To install libraries:
1. Simply run the terminal and go to the directory which contains the downloaded file
“install-libraries.sh”.
2. Run the command $: bash install-libraries.sh
3. Provide the password and wait for the libraries to be installed. 
ELSE USE:
Use the Following Commands:
sudo apt update
sudo apt install freeglut3-dev
sudo apt install glew-utils
sudo apt install libglew-dev
sudo apt install libfreeimage-dev
sudo apt make
sudo chmod +x game-release.cpp
make
./game-release
